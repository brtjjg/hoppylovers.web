# 💕 Hoppy Lovers

Full-featured social platform for hobby lovers to connect, share, and grow together.

## Features
- 🔐 User Authentication
- 👤 Profile with photos
- 🎯 AI-powered matching
- 💬 Messaging
- 📅 Events
- 🏘️ Communities
- 🔔 Notifications
- 🌙 Dark/Light theme
- 🛡️ Admin Panel
